# Keyboard Shortcuts

## Dashboard Keyboard Navigation

Speed up your workflow with keyboard shortcuts in the Sensei dashboard.

---

### Global Shortcuts

| Shortcut       | Action                       |
| -------------- | ---------------------------- |
| `?`            | Show keyboard shortcuts help |
| `/`            | Focus search bar             |
| `Esc`          | Close modal / Cancel action  |
| `Ctrl/Cmd + K` | Command palette              |

---

### Navigation

| Shortcut     | Action                 |
| ------------ | ---------------------- |
| `G` then `H` | Go to Home / Dashboard |
| `G` then `M` | Go to Migrations       |
| `G` then `C` | Go to Connections      |
| `G` then `S` | Go to Settings         |
| `G` then `L` | Go to Logs             |

---

### Migrations List

| Shortcut  | Action                  |
| --------- | ----------------------- |
| `N`       | New migration           |
| `J` / `↓` | Next migration          |
| `K` / `↑` | Previous migration      |
| `Enter`   | Open selected migration |
| `R`       | Refresh list            |

---

### Migration Detail

| Shortcut | Action                  |
| -------- | ----------------------- |
| `S`      | Start migration         |
| `P`      | Pause migration         |
| `X`      | Stop migration          |
| `V`      | View validation results |
| `L`      | View logs               |
| `E`      | Edit configuration      |
| `D`      | Duplicate migration     |

---

### Connections

| Shortcut | Action                     |
| -------- | -------------------------- |
| `N`      | New connection             |
| `T`      | Test selected connection   |
| `E`      | Edit selected connection   |
| `Delete` | Delete selected connection |

---

### Tables and Lists

| Shortcut       | Action           |
| -------------- | ---------------- |
| `J` / `↓`      | Move down        |
| `K` / `↑`      | Move up          |
| `Space`        | Toggle selection |
| `Ctrl/Cmd + A` | Select all       |
| `Enter`        | Open / Expand    |

---

### Modals and Forms

| Shortcut      | Action                       |
| ------------- | ---------------------------- |
| `Esc`         | Close / Cancel               |
| `Enter`       | Submit (when button focused) |
| `Tab`         | Next field                   |
| `Shift + Tab` | Previous field               |

---

### Editor (SQL/YAML)

| Shortcut               | Action           |
| ---------------------- | ---------------- |
| `Ctrl/Cmd + S`         | Save             |
| `Ctrl/Cmd + Z`         | Undo             |
| `Ctrl/Cmd + Shift + Z` | Redo             |
| `Ctrl/Cmd + F`         | Find             |
| `Ctrl/Cmd + H`         | Find and replace |
| `Ctrl/Cmd + /`         | Toggle comment   |
| `Ctrl/Cmd + D`         | Duplicate line   |

---

### Logs Viewer

| Shortcut       | Action                |
| -------------- | --------------------- |
| `F`            | Toggle follow mode    |
| `Ctrl/Cmd + F` | Search logs           |
| `G`            | Go to timestamp       |
| `E`            | Expand selected entry |
| `C`            | Copy selected entry   |

---

### Command Palette

Open with `Ctrl/Cmd + K`, then type:

| Command          | Action                 |
| ---------------- | ---------------------- |
| `>new migration` | Create new migration   |
| `>connections`   | Go to connections      |
| `>settings`      | Open settings          |
| `>help`          | Open documentation     |
| `>theme`         | Toggle dark/light mode |

---

### Customization

Keyboard shortcuts can be customized in Settings:

1. Go to **Settings** → **Preferences**
2. Click **Keyboard Shortcuts**
3. Click on any shortcut to reassign
4. Click **Reset to Defaults** to restore

---

### Tips

- Press `?` anytime to see available shortcuts for current view
- Shortcuts work when no input field is focused
- Most shortcuts use same patterns as popular tools (VS Code, Gmail)
